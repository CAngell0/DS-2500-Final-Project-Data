
# DS 2500 - Final Project Datasets

This is a repository that contains all the datasets used for my final project assignment in my DS 2500 class. I took this class in 2025 as a part of my Software Development bachelors degree program at the University of Utah.

#### List of Sources:
- Car Sales Dataset
    - Contans a couple thousand entries of car sales and their features
    - Contained in car_sales.csv
    - URL: https://www.kaggle.com/datasets/gagandeep16/car-sales
    - Credit: GaganBhatia
- Used Car Predictions
    - Listings of used cars along with their price and mileage
    - Contained in used_car_listings.csv
    - URL: https://www.kaggle.com/datasets/harikrishnareddyb/used-car-price-predictions
    - HariKrishnaReddyB
- DVM-CAR
    - A handful of different tables containing MSRP and other important information about car models.
    - Contained in the tables under ./DVM_CAR/
    - URL: https://deepvisualmarketing.github.io/
    - Jingming Huang, Bowei Chen, Lan Luo, Shigang Yue, and Iadh Ounis.
